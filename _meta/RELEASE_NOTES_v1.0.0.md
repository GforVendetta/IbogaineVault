# v1.0.0 — Initial Public Release

The IbogaineVault is a structured evidence map covering ~300 ibogaine research documents (1957–2026) converted to searchable markdown with standardised YAML metadata. Every document includes clinical classification, cross-references, and queryable fields enabling systematic evidence retrieval across seven decades of fragmented literature.

## What's Included

- **297 validated documents** across 6 categories (cardiac safety, clinical protocols, pharmacology/mechanisms, clinical outcomes, phenomenology, historical/policy)
- **Machine-readable index** — `papers.json` + `papers.csv` for programmatic access
- **Validation infrastructure** — `validate_vault.py` (schema compliance, wikilink resolution, duplicate DOI detection)
- **Three-layer navigation** — Bases (queryable databases), Hubs (curated domain synthesis), MOCs (researcher-centred maps)
- **Comprehensive governance documentation** — VAULT_PRINCIPLES.md, VAULT_ARCHITECTURE.md, schema_registry.yml, Tag_Taxonomy.md
